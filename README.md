# 🏨 Hotel Reservation Analytics Dashboard (Excel + DAX)

An interactive Excel-based analytics project designed to analyze **hotel reservation performance, revenue trends, customer behavior, and cancellation patterns** using Power Pivot, DAX, and Power Query.

The project is built on a **Star Schema Data Model**, enabling scalable and efficient analytical reporting.

---

## 🎯 BUSINESS OBJECTIVE

The objective of this project is to:
- Analyze hotel reservation performance and revenue trends  
- Monitor cancellation behavior and its impact on revenue  
- Understand customer segmentation and booking patterns  
- Improve decision-making through structured data modeling  

---

## 💡 SOLUTION

Developed an end-to-end Excel analytics solution that:
- Uses Power Query for data cleaning and transformation  
- Builds a Star Schema data model using Power Pivot  
- Implements DAX measures for KPIs and dynamic calculations  
- Provides multi-page interactive dashboards for insights  

---

## 🛠 TOOLS & TECHNOLOGIES

- Microsoft Excel (Power Pivot)  
- DAX (Data Analysis Expressions)  
- Power Query (ETL Process)  
- Data Modeling (Star Schema)  
- Data Visualization (Charts & Pivot Tables)

## 🔍 KEY INSIGHTS

- Significant revenue impact driven by cancellation behavior  
- Certain room types generate higher and more stable revenue  
- Market segment strongly influences booking patterns  
- Seasonal trends affect both bookings and cancellations  
- High-value customers contribute disproportionately to revenue

## 📈 BUSINESS IMPACT

- Improved visibility into hotel booking performance

- Enhanced customer segmentation and targeting strategy
